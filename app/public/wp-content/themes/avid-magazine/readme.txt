== Description ==
Avid Magazine WordPress Theme is very lightweight, easy to use, SEO friendly and customizable magazine theme. The theme is build in Bootstrap Framework. This is one of the best WordPress theme designed for lifestyle,  bloggers & blogging, reporters, magazine, newspaper, news portal and many more.

== Frequently Asked Questions ==

	= How to install the theme =
		* In your admin panel, go to Appearance > Themes and click the Add New button.
		* Click Upload and Choose File, then select the avid-magazine.zip file. Click Install Now.
		* Click Activate to use your new theme right away.
		* Go to Customiser -> Home Page Setting -> Select - Static Front Page
		* Create categories, pages and post
		* Select category needed for the homepage

== Installation ==
	
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the avid-magazine.zip file. Click Install Now.
3. Click Activate to use your new theme right away.
4. Go to Customiser -> Home Page Setting -> Select - Static Front Page
5. Create categories, pages and post
6. Select category needed for the homepage
